
Команды и клавиатурные привязки

Меню 'Файл':
Новый - нажмите 'Ctrl+N' чтобы создать новый файл
Открыть - нажмите 'Ctrl+O' чтобы открыть существующий файл
Открыть для чтения  - нажмите 'Ctrl+Shift+O' чтобы открыть файл только для чтения
Сохранить - нажмите 'Ctrl+S' чтобы сохранить открытый файл
Сохранить как - нажмите 'Ctrl+Shift+S' чтобы сохранить открытый файл как
Закрыть - нажмите 'Ctrl+Shift+C' чтобы закрыть открытый файл
Печать - нажмите 'Ctrl+P' чтобы распечатать открытый файл
Выйти - нажмите 'Alt+F4' чтобы выйти из программы

Меню 'Редактировать':
Вырезать - нажмите 'Ctrl+X' чтобы вырезать выделенный текст
Копировать - нажмите 'Ctrl+C' чтобы скопировать выделенный текст в буфер
Вставить - нажмите 'Ctrl+V' чтобы вставить текст из буфера
Удалить - нажмите 'Ctrl+D' чтобы удалить выделенный текст
Выделить все - нажмите 'Ctrl+A' чтобы выделить весь текст
Очистить - нажмите 'Ctrl+K' чтобы очистить весь текст

Меню 'Настройки':
Язык - выберите опцию чтобы изменить язык интерфейса
Привязки - выберите опцию чтобы изменить клавиатурную привязку
Тема - выберите опцию чтобы изменить внешнее оформление редактора

Меню 'Вид':
Проводник - нажмите 'Ctrl+E' чтобы отобразить проводник
МДИ - нажмите 'Ctrl+M' для использования многодокументного интерфейса

Меню 'Справка':
О программе - нажмите 'F12' чтобы посмотреть информацию о программе
Помощь - нажмите 'F1' чтобы показать окно помощи
